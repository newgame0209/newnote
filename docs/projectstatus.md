## 6. 開発環境のセットアップ

### バックエンド
```bash
# 1. 仮想環境の作成と有効化
python3 -m venv venv
source venv/bin/activate

# 2. 依存関係のインストール
pip install -r requirements.txt

# 3. 環境変数の設定
cp .env.example .env
# .envファイルを編集

# 4. データベースの初期化
python init_db.py

# 5. サーバーの起動
python app.py
```

### フロントエンド
```bash
# 1. 依存関係のインストール
npm install

# 2. 開発サーバーの起動
npm run dev
```
