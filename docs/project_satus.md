## 技術スタックの実装状況

### Fabric.js実装詳細
1. **導入方法**
   - CDNを使用して導入（jsdelivr経由）
   - `index.html`に以下のスクリプトを追加：
     ```html
     <script src="https://cdn.jsdelivr.net/npm/fabric@latest/dist/index.min.js"></script>
     ```

2. **キャンバス初期化**
   - `NoteEditor`コンポーネント内でFabric.jsキャンバスを初期化
   - `window.fabric`を使用してグローバルに利用可能なFabric.jsインスタンスにアクセス
   - キャンバスサイズは親要素に合わせて動的に設定

3. **描画ツール実装**
   - 各ツール（ペン、消しゴム、マーカー）に対して専用の設定関数を実装
   - `PencilBrush`を使用して描画機能を実現
   - ツール切り替え時に新しいブラシインスタンスを生成

4. **現在の課題**
   - テキスト入力機能の実装が必要
   - 画像挿入機能の実装が必要
   - ビューモードの実装が必要
   - 描画内容の保存機能の実装が必要
