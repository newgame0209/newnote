*   **プロジェクト名:** しゃべるノートアプリの MVP
*   **目標:** ノート作成と AI 音声変換の機能を統合した動作可能なプロトタイプを提供すること

### 技術スタック
- React + TypeScript
- Vite（開発環境）
- Fabric.js（キャンバス描画）
- TailwindCSS（スタイリング）

### 構築方法
1. 依存関係のインストール
   ```bash
   cd frontend
   npm install
   ```

2. 開発サーバーの起動
   ```bash
   npm run dev
   ```
   - ローカル環境: http://localhost:3000
   - ネットワーク環境: http://192.168.0.89:3000
